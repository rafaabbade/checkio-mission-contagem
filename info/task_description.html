Enunciado

Professor Thiago está organizando uma atividade de matemática para ajudar seus alunos a entender sequências numéricas e a prática de adicionar elementos a uma lista. Ele precisa de uma ferramenta que possa mostrar uma lista dos números de 1 até um número n escolhido pelo aluno. Escreva uma função que receba um número inteiro n e retorne uma lista contendo os números de 1 até n em ordem crescente. Use um loop para adicionar cada número à lista.

Utilize as dicas para verificar como criar e adicionar números à lista.

Entrada: Um número inteiro n.

Saída: Uma lista preenchida com números inteiros de 1 até n, em ordem crescente, impressos na tela.

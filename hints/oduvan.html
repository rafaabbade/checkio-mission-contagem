Vamos dividir em partes. Primeiramente, vamos aprender a criar uma lista. Não se preocupe com detalhes, pois ainda vamos praticar muito o uso das listas.

Criar uma lista vazia é muito parecido com criar uma variável, basta escolher um nome e atribuir colchetes vazios a ela: 

lista = [] # cria uma lista vazia

Legal, agora tenho uma lista vazia. Como posso adicionar os números?

Para adicionar números à lista, utilizamos o comando .append(). O comando é usado conforme indicado abaixo: 

lista = [] # cria uma lista vazia
lista.append(1) # o nome da lista sempre vem antes do .append() e o que iremos adicionar a lista vem sempre entre parênteses.

O comando .append() adiciona um elemento no final da lista.
